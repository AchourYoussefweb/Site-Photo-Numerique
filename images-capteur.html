Comment une image numérique est représentée

Une image numérique est composée de pixels. Chaque pixel peut être décrit avec trois valeurs : rouge, vert et bleu. Les intensités de ces trois canaux permettent d'obtenir un très grand nombre de couleurs.

Dans un appareil photo, la lumière arrive sur un capteur constitué de photosites. Chaque photosite transforme la lumière reçue en information numérique.

À retenir sur les images

  • Une image est un ensemble de pixels
  • Chaque pixel possède des canaux RVB
  • La définition indique le nombre de pixels
  • La résolution exprime les pixels par unité de longueur

Étapes du capteur photo

  1. La lumière entre dans l'appareil
  2. Les photosites reçoivent cette lumière
  3. Le signal lumineux est converti en signal électrique
  4. Le signal est numérisé et enregistré
